# franck.matsos.dev

Site personnel construit avec [Astro](https://astro.build/), TypeScript et TailwindCSS v4, prêt pour un déploiement automatique sur Netlify.

## Scripts
- `npm install` pour installer les dépendances.
- `npm run dev` pour lancer Astro avec la compilation Tailwind en mode surveillance.
- `npm run build` pour générer le CSS Tailwind puis produire le site statique dans `dist`.
- `npm run preview` pour tester le build localement.
- `npm run check` pour exécuter `astro check`.
