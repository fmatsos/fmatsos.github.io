# Suggested Commands

## Development
```bash
npm install          # Install dependencies
npm run dev          # Start dev server with Tailwind watch mode
npm run watch        # Alias for npm run dev
```

## Build & Preview
```bash
npm run build        # Build CSS then generate static site to dist/
npm run build:css    # Build Tailwind CSS only
npm run preview      # Preview the built site locally
```

## Type Checking
```bash
npm run check        # Run astro check for TypeScript/Astro validation
```

## Git Commands
```bash
git status           # Check working tree status
git add .            # Stage all changes
git commit -m "msg"  # Commit changes
git push             # Push to remote
```

## File System (Linux)
```bash
ls -la               # List files with details
find . -name "*.ts"  # Find TypeScript files
grep -r "pattern" .  # Search for pattern in files
```
